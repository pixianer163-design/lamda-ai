# Agent Factory

基于元学习(DPML)的下一代量化交易Agent平台。

## 项目定位

Agent Factory是一个可扩展的多Agent量化交易平台，支持：
- 多策略并行运行（A/B测试）
- 元学习驱动的世界模型
- DSL策略描述语言
- 云端训练与边缘推理

## 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                    Agent Factory                        │
├─────────────────────────────────────────────────────────┤
│  用户交互层 → Agent编排 → Agent实例 → 数据持久化        │
├─────────────────────────────────────────────────────────┤
│  World Model 2.0 (DPML)                                 │
│  ├─ 显式记忆 (LSTM) - 长期规律                          │
│  ├─ 隐式记忆 (Transformer) - 短期适应                   │
│  └─ 快速适应 (5步梯度下降)                              │
├─────────────────────────────────────────────────────────┤
│  多Agent并行                                            │
│  ├─ Agent A: 基础版 (多因子策略)                        │
│  ├─ Agent B: 保守版 (价值投资)                          │
│  └─ Agent C: 激进版 (动量策略)                          │
├─────────────────────────────────────────────────────────┤
│  智能融合层                                             │
│  └─ 动态权重: 元学习信号 + 规则信号 + 策略信号          │
├─────────────────────────────────────────────────────────┤
│  DSL策略引擎 (规划中)                                   │
│  ├─ 类Python语法                                        │
│  ├─ 自然语言→DSL (LLM翻译)                              │
│  └─ 可视化拖拽                                          │
└─────────────────────────────────────────────────────────┘
```
